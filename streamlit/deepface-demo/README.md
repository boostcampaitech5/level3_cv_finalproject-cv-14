# streamlit 사용 방법

## Installation

```shell
$ pip install streamlit
```

```shell
$ pip install deepface
```

```shell
$ pip install streamlit-option-menu
```

## Run

```shell
$ streamlit run app.py --server.port {포트번호}
```

External URL 접속
